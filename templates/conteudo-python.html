Estruturas de seleção

Em Python, as estruturas de seleção permitem que o programa execute diferentes blocos de código com base em condições. As principais estruturas de seleção são o if, elif e else.

Estrutura 'if'

A estrutura if é usada para executar um bloco de código se uma condição for verdadeira. A sintaxe é a seguinte:

if condição:
# Código a ser executado se a condição for verdadeira

Estrutura 'elif'

A estrutura elif (abreviação de 'else if') é usada para verificar múltiplas condições. Se a condição do if for falsa, o programa verifica a condição do elif. A sintaxe é a seguinte:

elif outra_condição:
# Código a ser executado se a outra_condição for verdadeira

Estrutura 'else'

A estrutura else é usada para executar um bloco de código se nenhuma das condições anteriores (if ou elif) for verdadeira. A sintaxe é a seguinte:

else:
# Código a ser executado se nenhuma condição for verdadeira

Exemplo Completo

Vamos ver um exemplo prático de como essas estruturas funcionam juntas:

idade = 20

if idade >= 18:
print("Você é maior de idade.")
elif idade >= 13:
print("Você é adolescente.")
else:
print("Você é criança.")

Estruturas de repetição

Estruturas de repetição permitem executar um bloco de código várias vezes. Em Python, as principais são for e while.

Estrutura 'for'

for i in range(5):
print(i)

Estrutura 'while'

while condição:
print("Executando...")

Vetores e matrizes

Vetores (listas) e matrizes (listas de listas) são usados para armazenar coleções de dados. Exemplo de vetor:

numeros = [1, 2, 3]

Exemplo de matriz:

matriz = [[1, 2], [3, 4]]

Funções e procedimentos

Funções permitem reutilizar código. Exemplo:

def saudacao():
print("Olá!")

Tratamentos de exceção

O tratamento de exceções permite lidar com erros de execução. Exemplo:

try:
print(1/0)
except ZeroDivisionError:
print("Não é possível dividir por zero.")
